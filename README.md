# Diagnostics Project
This project provides a diagnostics system with a feedback form and review workflow.

## Usage
1. Create a diagnostics instance using `create_diagnostics()`.
2. Provide feedback using `provide_feedback()` method.
3. Store feedback using `store_feedback()` method.
4. Get feedback using `get_feedback()` method.
5. Trigger review workflow using `review_workflow()` method.

## Testing
Run tests using `pytest` command.
