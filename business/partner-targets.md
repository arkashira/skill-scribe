# Partner Targets
## Introduction
To enhance the value proposition of skill-scribe, we aim to integrate with complementary SaaS/APIs, focusing on those that offer affiliate or revenue-share programs. This document outlines our partner integration roadmap, highlighting 7 key targets.

## Partner Integration Roadmap
The following SaaS/APIs are prioritized based on their potential to add value to skill-scribe users, with a focus on affiliate/revenue-share opportunities:

1. **GitHub**
	* Free-tier limits: 1000 API requests per hour
	* Integration effort: M
	* Value-add: Streamlines code analysis and feedback by leveraging GitHub's version control and collaboration features
	* Affiliate/revenue-share: GitHub Developer Program (up to 15% revenue share)
2. **Codiga**
	* Free-tier limits: 100 code analyses per month
	* Integration effort: S
	* Value-add: Enhances code review and optimization capabilities for skill-scribe users
	* Affiliate/revenue-share: Codiga Affiliate Program (10% revenue share)
3. **Repl.it**
	* Free-tier limits: 100MB storage, 1000 requests per day
	* Integration effort: M
	* Value-add: Enables seamless code execution and testing for skill-scribe users
	* Affiliate/revenue-share: Repl.it Affiliate Program (15% revenue share)
4. **CodePen**
	* Free-tier limits: 100 pens, 1000 views per month
	* Integration effort: S
	* Value-add: Expands skill-scribe's front-end development capabilities and community engagement
	* Affiliate/revenue-share: CodePen Affiliate Program (10% revenue share)
5. **Pluralsight**
	* Free-tier limits: Limited access to courses and tutorials
	* Integration effort: L
	* Value-add: Provides skill-scribe users with access to a vast library of developer courses and tutorials
	* Affiliate/revenue-share: Pluralsight Affiliate Program (up to 50% revenue share)
6. **GitLab**
	* Free-tier limits: 1000 API requests per hour
	* Integration effort: M
	* Value-add: Offers an alternative to GitHub, with a focus on enterprise and self-hosted solutions
	* Affiliate/revenue-share: GitLab Affiliate Program (10% revenue share)
7. **Stack Overflow**
	* Free-tier limits: Limited access to premium features
	* Integration effort: L
	* Value-add: Integrates skill-scribe with the largest online community of developers, enhancing user support and resources
	* Affiliate/revenue-share: Stack Overflow Affiliate Program (10% revenue share)

## Prioritization
The prioritization of these integrations is based on the following factors:

* Affiliate/revenue-share opportunities
* Value-add to skill-scribe users
* Integration effort and complexity
* Market demand and competition

By focusing on these key integrations, we can enhance the value proposition of skill-scribe, drive user engagement, and generate revenue through affiliate and revenue-share programs.
