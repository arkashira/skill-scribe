# breakeven.md
## Unit Economics
### Cost per Active User
The estimated costs for running the skill-scribe application are as follows:
- Compute: $0.05 per user per month (assuming 10 hours of compute time per month at $0.005 per hour)
- Storage: $0.01 per user per month (assuming 100MB of storage per user at $0.10 per GB)
- Bandwidth: $0.02 per user per month (assuming 1GB of bandwidth per user at $0.02 per GB)
Total cost per active user: $0.08 per month

## Pricing Tiers
The following pricing tiers are proposed for the skill-scribe application:
1. **Basic**: $9.99 per month - Limited to 100 code analyses per month, basic feedback and suggestions
2. **Pro**: $29.99 per month - Unlimited code analyses, advanced feedback and suggestions, priority support
3. **Enterprise**: $99.99 per month - All Pro features, plus dedicated support, customizable feedback and suggestions, on-premises deployment option

## Customer Acquisition Cost (CAC) Range
The estimated CAC range for the skill-scribe application is between $10 and $50 per user, depending on the marketing channels and strategies used.

## Lifetime Value (LTV) Estimate
The estimated LTV for a skill-scribe user is $200, assuming an average subscription duration of 6 months and an average revenue per user (ARPU) of $33.33 per month.

## Break-even Users Count
To calculate the break-even point, we need to consider the CAC and LTV estimates. Assuming a CAC of $30 per user and an LTV of $200 per user, the break-even point would be:
Break-even point = CAC / (LTV - CAC) = $30 / ($200 - $30) = 0.17 or 17% of users
However, to calculate the break-even users count, we need to consider the monthly costs and revenues. Assuming 1000 users, the monthly revenue would be:
- Basic: 300 users \* $9.99 = $2997
- Pro: 400 users \* $29.99 = $11996
- Enterprise: 300 users \* $99.99 = $29997
Total monthly revenue = $2997 + $11996 + $29997 = $44990
Monthly costs = 1000 users \* $0.08 = $80
Break-even users count = (CAC + monthly costs) / (ARPU - monthly costs per user)
Assuming an average ARPU of $44.99 and an average monthly cost per user of $0.08, the break-even users count would be:
Break-even users count = ($30 + $80) / ($44.99 - $0.08) = 1300 users

## Path to $10K MRR
To reach $10K MRR, we would need:
- 1000 users on the Basic tier: 1000 users \* $9.99 = $9990 (short of the target)
- 334 users on the Pro tier: 334 users \* $29.99 = $10013 (meets the target)
- 100 users on the Enterprise tier: 100 users \* $99.99 = $9999 (short of the target)
Therefore, the path to $10K MRR would be to acquire 334 users on the Pro tier.
